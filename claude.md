# Claude Open Source Program Application

## Project
**HireLive (Interview Platform)**

Live demo: https://hirelive.onrender.com

HireLive is an open-source full-stack interview practice platform where developers can run realistic technical interview sessions with live coding, collaborative session workflows, and integrated communication features.

## What the project does
- User authentication and protected routes with Clerk
- Interview session lifecycle: create, join, and end sessions
- Active and recent session tracking
- Realtime communication integrations (Stream Chat and Stream Video)
- MongoDB persistence for users and sessions
- Event-driven user synchronization using Inngest

## Tech stack
- Frontend: React, Vite, React Router, React Query, Tailwind CSS, DaisyUI
- Backend: Node.js, Express, Mongoose
- Auth: Clerk
- Realtime: Stream Chat + Stream Video
- Database: MongoDB
- Events: Inngest

## Open-source goals
- Help students and early-career developers practice interview skills in a realistic environment
- Provide a clean reference architecture for realtime full-stack apps
- Continue improving collaboration and evaluation workflows for mock interviews
